# emeraldjava-hugo

A hugo static site for emeraldjava.

Steps

1. hugo new site . --force
2. git clone https://github.com/lxndrblz/anatole.git themes/anatole
3. hugo server -D
4. hugo --minify

## themes

### hyde

    git submodule add https://github.com/spf13/hyde.git themes/hyde
